# RPJ_SOC
Celý Unity Project.
Na jeho otvorenie je potrebné mať nainštalovanú aplikáciu Unity Hub. Po otvorení v Editore Unity treba otvoriť ľubovoľnú scénu v priečinku assets/scenes.

V prípade záujmu pozrieť si len zdrojové kódy bez otvárania celého projektu - všetky skripty s príponou .cs sú umiestnené v priečinku 2020_2021/assets/scripts.
